In HTML Template Binding
{{ orderBy_expression | orderBy : expression : reverse}}

In JavaScript
$filter('orderBy')(array, expression, reverse)
Code Example
orderBy_expression x in names (set in table row where data is to be displayed)
orderBy orderBy
expression (HTML/JavaScript) (1)orderByField e.g. orderByField='name' (set in table header) (2) An expression can be optionally prefixed with + or - to control ascending or descending sort order (for example, +name or -name). (3) You can initiate sorting in code like $scope.variableName = '-name'; Write only [variableName] where expression is to be used in table.
reverse reverseSort = !reverseSort (set in table header i.e. reverse the order of the array. ! means not and creates toggle effect)
array The array to sort.
